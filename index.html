Understand Website Loading Slowness

This lets a web developer:
  • Optimize file load patterns.
  • Find undexpected delays.
  • Track performance progress.
  • ... Including when JS frameworks build it after onLoad.
Questions:
Why only newer Chrome & IE?
They implement performance timeline.
Why exactly 150 entries, with early ones missing?
Fix it by adding at onload the Javascript (for Chrome): performance.webkitSetResourceTimingBufferSize(10000)
Some resources only show Fetch, others have more data
Other domain servers should set the header of Timing-Allow-Origin: *
